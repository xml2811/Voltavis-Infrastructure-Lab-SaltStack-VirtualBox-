Nuestro Compromiso Social

Talleres web gratuitos para jóvenes
Organizamos talleres prácticos sobre diseño y creación de páginas web, dirigidos a estudiantes y jóvenes apasionados por la tecnología.
Una página web gratuita al año
Cada año seleccionamos un proyecto con impacto social al que le regalamos una web profesional totalmente gratuita.
Actividades comunitarias
Participamos en jornadas locales de limpieza y mejora de espacios públicos para contribuir con nuestro entorno.
Reciclaje de equipos tecnológicos
Recogemos, reparamos y donamos equipos tecnológicos a colectivos con menos recursos para cerrar la brecha digital.
Plantillas web gratuitas
Creamos plantillas web modernas y gratuitas que regalamos mediante concursos abiertos a cualquier participante.

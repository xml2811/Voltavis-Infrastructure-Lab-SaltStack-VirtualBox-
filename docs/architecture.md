# Architecture

## English

This lab simulates a Linux infrastructure composed of several services managed through SaltStack.

Main components:

- SaltStack control node.
- Debian servers.
- Apache web service.
- SFTP access.
- Remote access through OpenVPN.
- DNS and DHCP services.
- Firewalling and network segmentation.
- LAN/DMZ concepts.
